Text Analysis Project
This repository contains a Python script for analyzing text documents, specifically designed for educational purposes as part of an assignment from boot.dev. The project focuses on analyzing Mary Shelley's "Frankenstein," counting the frequency of each letter and the total number of words in the text.

Project Description
The script performs the following tasks:

Reads the entire text of "Frankenstein" from a text file.
Counts the total number of words in the document.
Counts the frequency of each letter in the document (case-insensitive and alphabetic characters only).
Displays a report summarizing the findings, including the total word count and the frequency of each letter, sorted by frequency in descending order.
